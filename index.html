Paradigma de la OOP

La programación orientada a objetos (POO) se basa en el concepto de crear un modelo del problema que se desea resolver en forma de objetos en los programas. Esta paradigma de programación disminuye los errores y promueve la reutilización del código. Python es un lenguaje que soporta la programación orientada a objetos.
Los objetos definidos en Python cuentan con las siguientes características:
Encapsulación: Los datos y métodos de un objeto están encapsulados dentro de él, lo que oculta los detalles de implementación y proporciona un interfaz bien definida para interactuar con el objeto.
Herencia:Los objetos pueden heredar propiedades y comportamientos de otros objetos, lo que facilita la reutilización de código y la creación de jerarquías de clases.
Polimorfismo: Los objetos de diferentes clases pueden compartir el mismo interfaz, lo que permite escribir código genérico que funcione con objetos de diferentes tipos.
Abstracción: Los objetos ocultan los detalles de implementación internos y exponen solo la información necesaria para interactuar con ellos, lo que simplifica el diseño y el uso de los programas.
